Filing by phone (TELEFILE) – File your GST/HST return

GST/HST TELEFILE is a service that allows qualifying registrants to file their GST/HST returns directly to the Canada Revenue Agency (CRA) over the internet using their touch-tone telephone.

On this page

  • What you can and cannot do
  • Before you start
  • How to file

What you can and cannot do

Most GST/HST registrants across Canada and the United States can file a regular return using this service.

You can:

  • Meet electronic filing requirements
  • Get a confirmation number right away
  • File without an online CRA account

You cannot:

  • File without an GST/HST access code
  • File elections
  • File a return for an account administered by Revenu Québec
  • File a combined GST/HST and QST return for selected listed financial institutions (form RC7200)
  • File a return for a non-resident
  • File a simplified GST/HST return for a non-resident digital economy business
  • Include schedules
  • File Information on Claims Paid or Credited for Foreign Conventions and Tour Packages (form GST106)
  • File rebate (refer to Claim a GST/HST rebate)

If you want to use TELEFILE for your return but need to file a rebate form, include the rebate amount on your electronic return and mail the application form to the applicable tax centre. Make sure the tax centre receives the form on or before the due date of your return.

If you cannot file what you need to using this method, you can choose another filing method that works for you.

Before you start

If you plan to file your return by phone using GST/HST TELEFILE, make sure you:

  • Are using a touch-tone phone
  • Know what you can and cannot file using this service
  • Have your four-digit access code
  • Have your Business Number (BN)
  • Have your reporting period "from" and "to" dates
  • Prepared your return information in Canadian funds
  • Are aware of the hours of service for CRA electronic services

How to file

  1. To use GST/HST TELEFILE to file your return, dial the toll-free telephone number 1-800-959-2038
    • An automated system will guide you through the process. The system will ask you to enter your tax information using your telephone keypad
  2. Use the following keys on your touch-tone phone to make selections during the filing process:
    • Press "*" to have a message repeated
    • Press "1" to answer yes or "0" to answer no
    • Press "#" when you have finished entering an amount
  3. Enter the requested information
    • Enter all the digits of your Business Number, but do not enter the letters "RT"
    • Enter... Can we say what else need to tell them?
  4. The GST/HST TELEFILE system repeats each entry to help you file an accurate return
    • Make any necessary corrections before you confirm your amounts
  5. Once the data has been entered, you will be asked to confirm that the information you entered is correct
    • If you answer yes, the CRA will consider your return received
    • If you hang up at any time before confirming that you want to file your return, the information you entered will not be saved
    • You can hang up and call again up to a maximum of six times
  6. The automated system will immediately issue you a six-digit confirmation number
    • Write down your confirmation number and keep it with your records
    • Your return will then begin processing
  7. Depending on whether you expect to make a payment or receive a refund, review the following information:
    1. If you have an amount owing, find out how to make a payment to the CRA
    2. Consider setting up direct deposit to speed up any refund you may be expecting
