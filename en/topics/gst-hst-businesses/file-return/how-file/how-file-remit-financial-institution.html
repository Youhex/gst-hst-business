How to file and remit though a financial institution – File your GST/HST return

GST/HST registrants can electronically file their returns and pay any amounts owing at the same time through a participating Canadian financial institution. The institution transmits the return in a standard format to the Canada Revenue Agency (CRA) through a system called GST/HST Electronic Data Interchange (EDI).

Some financial institutions may charge a fee for this service.

What you can and cannot do

Most GST/HST registrants can upload a regular return (GST34) using this service.

You can:

  • Meet electronic filing requirements
  • Get a confirmation number right away
  • Include a payment when filing
  • Include certain schedules (this depends on the financial institution)
    • Schedules B and C, to recapture input tax credits for the provincial portion of the HST on specified property or services)
  • File a return for a non-resident

Filing a GST/HST return for a non-resident

You must:

  • Complete all GST/HST amounts in Canadian dollars
  • File using a participating financial institution in Canada
  • Remit Canadian funds using a participating financial institution in Canada

You cannot:

  • File a return for an account administered by Revenu Québec
  • File form RC7200, combined GST/HST and QST return for selected selected listed financial institutions
  • File a simplified GST/HST return for a non-resident digital economy business
  • Include Schedule A, for builders to report transitional information on housing
  • File elections
  • File rebates with your return (refer to Claim a GST/HST rebate):
    • Form GST66, Application for GST/HST Public Service Bodies' Rebate and Self-Government Refund
    • Form GST189, general GST/HST rebate application (all reason codes)
      • Form GST189 (reason code 23), Ontario First Nations point-of-sale relief
    • Form GST190, New Housing Rebate Application for Houses Purchased from a Builder
    • Form GST490, GST/HST Rebate Application for Federal, Provincial, and Territorial Governments
    • Form GST524, New Residential Rental Property Rebate Application
  • File form GST106, Information on Claims Paid or Credited for Foreign Conventions and Tour Packages

If you want to file your return through a financial institution but need to file a rebate form, include the rebate amount on your electronic return and mail the application form to the applicable tax centre. If you cannot file what you need to using this method, you can choose another filing method that works for you.

Using the service

  1. Register with a participating Canadian financial institution
  2. Prepare your return information in Canadian funds
  3. Provide the financial institution with the necessary GST/HST return and any payment information through your computer, by phone, or any other mutually agreeable method
  4. The financial institution will convert the information into a return in the required EDI format and will transmit the return and any payment to the CRA
