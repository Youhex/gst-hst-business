Filing and remitting though a financial institution – File your GST/HST return

GST/HST registrants can electronically file their returns and pay any amounts owing at the same time through a participating Canadian financial institution. The institution transmits the return in a standard format to the Canada Revenue Agency (CRA) through a system called GST/HST Electronic Data Interchange (EDI).

Some financial institutions may charge a fee for this service.

What you can and cannot do

Most GST/HST registrants can file a regular return using this service.

You can:

  • Meet electronic filing requirements
  • Get a confirmation number right away
  • File without an GST/HST access code
  • File without an online CRA account
  • Include schedules B and C
  • File a return for a non-resident

Filing a GST/HST return for a non-resident through a financial institution

You must:

  • Complete all GST/HST amounts in Canadian dollars
  • File using a participating financial institution in Canada
  • Remit Canadian funds using a participating financial institution in Canada

You cannot:

  • File elections
  • File a return for an account administered by Revenu Québec
  • File a combined GST/HST and QST return for selected listed financial institutions (form RC7200)
  • File a simplified GST/HST return for a non-resident digital economy business
  • Include Schedule A
  • File Information on Claims Paid or Credited for Foreign Conventions and Tour Packages (form GST106)
  • File rebates (refer to Claim a GST/HST rebate)

If you cannot use EDI

Choose another filing method that works for you.

If you want to file your return through a financial institution but need to file an additional form or rebate claim, you can include the amount on your electronic return (Is this true for EDI?) and mail the form to the applicable tax centre. Make sure the tax centre receives the form on or before the due date of your return.

Before you start

If you plan to file your return and remit any tax through GST/HST Electronic Data Interchange (EDI), make sure you:

  • Know what you can and cannot file using this service
  • Have your Business Number (BN)
  • Have your reporting period "from" and "to" dates
  • Have a web browser and browser settings required by your financial institution

Using the service

  1. Register with a participating Canadian financial institution
  2. Prepare your return information in Canadian funds
  3. Provide the financial institution with the necessary GST/HST return and any payment information through your computer, by phone, or any other mutually agreeable method
    • The financial institution will convert the information into a return in the required EDI format and will transmit the return and any payment to the CRA
    • The transfer of funds is guaranteed on the value date you establish with your financial institution
  4. If you are expecting a refund, consider setting up direct deposit (Is DD needed for refund claimed through EDI?)
    • Your GST/HST refund can be deposited directly into your account within 10-15 calendar days
  5. You can sign up for email notifications from the CRA by providing your email address
    • We will then send you an email when notices, letters, and statements are available to be viewed in your CRA account instead of mailing you a paper copy
    • A separate registration for CRA account is required

Contact

If you require assistance with filing your return using EDI, contact your financial institution.
