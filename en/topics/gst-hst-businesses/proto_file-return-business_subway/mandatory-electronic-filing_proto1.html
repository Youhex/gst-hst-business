Mandatory electronic filing for GST/HST registrants

For reporting periods that begin after December 31, 2023, all GST/HST registrants, with the exception of charities and selected listed financial institution (SLFIs), are required to file returns electronically. If you continue to file a paper return, you will be charged a penalty.

Find out about the ways to file electronically

If you are unable to file electronically

If you are unable to file electronically even though you are not a charity or SLFI, you can request an exemption. Exemptions are granted on a case-by-case basis only.

To request an exemption, please write us a letter explaining the reasons you cannot comply with mandatory electronic filing. Mail the letter to:
