Intro here

View a previously filed GST/HST return

You may view the status and the details of a previously filed GST/HST return using My Business Account. You will also be able to view upcoming and overdue GST/HST returns by using the same digital service.

Changing a return already filed

Returns can only be adjusted electronically using the My Business Account (MyBA) portal. If you do not already have a MyBA user ID and password, you can create one at My Business Account.

If you are not registered with MyBA, send your adjustment request, in writing, to your tax centre.
