Question: Is the GST/HST access code a different code than the access code used for Individual filing? Is not, something should be explaied about it here.

The GST/HST access code is a four-digit access code is required for some of GST/HST filing methods. This code acts as a personal security key that authenticates you when you are not logged into a CRA account. It should not be shared with unauthorized individuals.

Your GST/HST access code will remain the same for all filing periods unless you change it..

Filing methods that do not use an access code

  • GST/HST NETFILE (accessed through the "File a return" link in My Business Account)
  • GST/HST Electronic Data Interchange (EDI)
  • Paper filing

Filing methods that use an access code

  • GST/HST Internet File Transfer
  • GST/HST TELEFILE
  • GST/HST NETFILE (accessed though the web form only)

Obtaining an access code

  • If you are a new user and have never received an access code, call 1-800-959-5525.
  • If calling from outside Canada and the United States, call collect at 613-940-8497.

Changing an access code

  • Business owners can change their access code online using the GST/HST Access Code Online service.
  • The access code can also be created or changed over the phone by the business owner or an authorized representative with level 2 authorization or higher.

When an accountant can use your access code to telefile (does it belong here?)

If your accountant is legally authorized to sign the paper copy of your GST/HST return, they can use GST/HST TELEFILE to file your return on your behalf. They will need to use your GST/HST access code.

If your accountant is not authorized to file GST/HST returns on your behalf, you can sign up for My Business Account and add your accountant as an authorized representative.
