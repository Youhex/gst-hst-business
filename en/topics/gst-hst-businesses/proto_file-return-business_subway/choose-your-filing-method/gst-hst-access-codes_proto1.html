The GST/HST access code is a four-digit access code is required for some GST/HST filing methods. It acts as a personal security key that authenticates you when you are not logged into a CRA account, and should not be shared with unauthorized individuals.

Your GST/HST access code will remain the same for all filing periods unless you change it.

When you need an access code

You do not always need an access code for filing electronically.

You need an access code if you are filing:

  • Online using the NETFILE form (outside of a CRA account)
  • Using third-party accounting software (GST/HST Internet File Transfer)
  • Over the phone using GST/HST TELEFILE

You do not need an access code if you are filing:

  • In your online CRA account (My Business Account or Represent a Client)
  • Through a participating Canadian financial institution
  • By mail

A representative that is legally authorized to sign the paper copy of your GST/HST return may also use your access code to file on your behalf.

How to get or change an access code

If you lose or forget your access code, or want to change your access code for other reasons, you can get a new code online.

If you are a new GST/HST registrant and have never received an access code, phone us to get an access code. An existing access code can also be changed over the phone.

Online

You can set a new access code using GST/HST Access Code Online.

Your new access code can be a number you choose, and you can use it to file all future GST/HST returns.

To change your GST/HST access code, you will need to enter your business number and information from a previously filed return, either:

  • The reporting period and line 109 (net tax) where the amount on line 109 is not $0.00
  • The reporting period and confirmation number from a return filed using GST/HST NETFILE, TELEFILE, or Internet File Transfer

By phone
