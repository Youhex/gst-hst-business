GST/HST registrants can electronically file their returns and pay any amounts owing at the same time through a participating Canadian financial institution. The institution transmits the return in a standart format to the Canada Revenue Agency (CRA) through a system called GST/HST Electronic Data Interchange (EDI). You can use EDI to file the following:

  • Form GST34, Goods and Services Tax/Harmonized Sales Tax (GST/HST) Return for Registrants
  • Schedule B for the recapture of input tax credits
  • Schedule C for the reconciliation of recaptured input tax credits

Previous intro: EDI is a computer-to-computer exchange of information in a standard format. You can use EDI to file the electronic equivalent of Form GST34, Goods and Services Tax/Harmonized Sales Tax (GST/HST) Return for Registrants, Schedule B for the recapture of input tax credits, and Schedule C for the reconciliation of recaptured input tax credits. You can also pay the net tax you owe electronically through a participating Canadian financial institution.

How it works

  • You need to register with a participating financial institution
  • You can then provide the necessary GST/HST return and payment information to them through your computer, by phone, or any other mutually agreeable method
  • The financial institution converts the information into an electronic GST/HST return and/or payment that meets the Canada Revenue Agency (CRA)'s specifications and transmits the return and/or payment to the CRA
  • Some financial institutions may charge a fee for this service

Who can file using EDI

GST/HST EDI is available to GST/HST registrants across Canada, excluding those that are administered by Revenu Qu√©bec.

You can file your return using EDI if you are:

  • A non-resident
    • You must file using a participating financial institution in Canada
    • You must complete all GST/HST amounts in Canadian dollars
    • You must remit Canadian funds to the CRA using a participating financial institution in Canada

You cannot file your return using EDI if you are:

  • A builder required to report transitional information on housing (Schedule A of the GST/HST return)
    • You must file your return using the your online CRA account or the GST/HST online NETFILE form

Benefits of using EDI

There are advantages to filing your return using EDI whether you are receiving a refund or making a payment.

  • Accuracy - you key in the information, eliminating transcription errors from further handling
  • Cash flow control - transfer of funds is guaranteed on the value date you establish with your financial institution
  • Quicker refund payments - your GST/HST refund can be deposited directly into your account within 10-15 calendar days
  • Convenience - EDI ends the need to mail a return
