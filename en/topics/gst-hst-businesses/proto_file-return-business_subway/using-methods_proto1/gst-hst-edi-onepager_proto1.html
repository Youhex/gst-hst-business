GST/HST registrants are able to electronically file their returns and pay any amounts owing at the same time through participating Canadian financial institutions. The return is transmitted from the bank to the Canada Revenue Agency (CRA) through a system called GST/HST Electronic Data Interchange (EDI), a computer-to-computer exchange of information in a standard format. You can use this method to:

  • File the electronic equivalent of Form GST34, Goods and Services Tax/Harmonized Sales Tax (GST/HST) Return for Registrants
  • Schedule B for the recapture of input tax credits
  • Schedule C for the reconciliation of recaptured input tax credits
  • Pay the net tax you owe

You can also pay the net tax you owe electronically through a participating Canadian financial institution.

How it works

  • You need to register with a participating financial institution
  • You can then provide the necessary GST/HST return and payment information to them through your computer, by phone, or any other mutually agreeable method
  • The financial institution converts the information into an electronic GST/HST return and/or payment that meets the Canada Revenue Agency (CRA)'s specifications and transmits the return and/or payment to the CRA
  • Some financial institutions may charge a fee for this service

Benefits of using EDI

There are advantages to filing your return using EDI whether you are receiving a refund or making a payment.

  • Accuracy - you key in the information, eliminating transcription errors from further handling
  • Cash flow control - transfer of funds is guaranteed on the value date you establish with your financial institution
  • Quicker refund payments - your GST/HST refund can be deposited directly into your account within 10-15 calendar days
  • Convenience - EDI ends the need to mail a return

Builders - transitional information

If you are a builder required to report transitional information on housing (Schedule A of the GST/HST return), you must file your return using GST/HST NETFILE.

Recaptured input tax credits

GST/HST EDI can be used to file returns with Schedule B (recapture of input tax credits) and Schedule C (reconciliation of recaptured input tax credits). Contact your financial institution to see if they offer these filing options.

Non-residents filing GST/HST returns

Non-residents can file their GST/HST returns using EDI. However, they must complete all GST/HST amounts in Canadian dollars and remit Canadian funds to the CRA using a participating financial institution in Canada.
