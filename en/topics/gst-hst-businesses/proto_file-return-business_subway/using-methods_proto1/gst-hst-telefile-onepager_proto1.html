GST/HST TELEFILE is a service that allows qualifying registrants to file their GST/HST returns directly to the Canada Revenue Agency (CRA) over the internet using their touch-tone telephone.

On this page

  • What you can and cannot do
  • Before you start
  • How to file
  • Notices

Who can file using TELEFILE

GST/HST TELEFILE is available to GST/HST registrants across Canada and the United States, excluding those with accounts administered by Revenu Québec.

The following registrants are not eligible to use GST/HST TELEFILE and must use an alternate filing method:

  • Registrants whose accounts are administered by Revenu Québec
  • Builders required to report transitional information on housing (Schedule A of the GST/HST return)
  • Registrants required to recapture input tax credits for the provincial portion of the HST on specified property or services (Schedules B and C of the GST/HST return)

Does this need to be here? If you are a selected listed financial institution (SLFI) that reports both the GST/HST and QST, you must determine which returns to complete and when to file.

Before you start

Before using GST/HST TELEFILE, complete the following steps:

  • Verify that you are eligible to use GST/HST TELEFILE
  • Ensure you have a touch-tone phone
  • Review the hours of service for CRA electronic services
  • Ensure you have a four-digit access code
  • Prepare your return information in Canadian funds
  • If you have an amount owing, find out how to make a payment to the CRA

Using the service

  1. To use GST/HST TELEFILE to file your return, dial the toll-free telephone number 1-800-959-2038. An automated system will guide you through the process. The system will ask you to enter your tax information using your telephone keypad.
    • Enter all the digits of your Business Number, but do not enter the letters "RT"
  2. Use the following keys on your touch-tone phone to make selections during the filing process:
    • Press "*" to have a message repeated.
    • Press "1" to answer yes or "0" to answer no.
    • Press "#" when you have finished entering an amount.
  3. The GST/HST TELEFILE system repeats each entry to help you file an accurate return. Make any necessary corrections before you confirm your amounts.
  4. Once the data has been entered, you will be asked to confirm that the information you entered is correct. If you answer yes, the CRA will consider your return received.
    • If you hang up at any time before confirming that you want to file your return, the information you entered will not be saved. You can hang up and call again up to a maximum of six times.
  5. The automated system will immediately issue you a six-digit confirmation number. Write down your confirmation number and keep it with your records. Your return will then begin processing.
  6. Consider setting up direct deposit to speed up any refund you may be expecting.
